# aeroo_space_AI_NeuroSpace
FlaringWatch CA3 - интерактивный мониторинг газового факельного сжигания в Центральной Азии (KAZ, UZB, TKM) по спутниковым данным за 2012-2019: карта, кластеризация, топ-alerts.
FlaringWatch CA3 - это минимально воспроизводимый workflow для анализа спутниковых детекций газовых факелов в Центральной Азии (Казахстан, Узбекистан, Туркменистан) за 2012-2019 гг. На основе параметров детекций выполняется кластеризация объектов и строится интерпретируемая шкала воздействия (Low/Medium/High), после чего формируются top-alerts (наиболее значимые объекты) и интерактивная карта. Решение ориентировано на демонстрацию полного пайплайна: данные -> признаки -> кластеризация -> impact-class -> alerts -> web-карта.

Что внутри (AI / аналитика)

В проекте применяется кластеризация (KMeans) по признакам спутниковых детекций, чтобы разделить объекты на группы с различающимся “профилем” активности. Затем каждой группе назначается интерпретируемый класс воздействия impact_class (Low/Medium/High) на основе агрегированного impact_score, вычисленного из нормированных признаков. После этого формируется таблица top-alerts (наиболее аномальные/значимые объекты) и интерактивная карта с переключаемыми слоями по годам и по классу воздействия.

Используемые признаки (на уровне объекта/детекции):

flr_volume_log1p (лог-преобразование объема/интенсивности факела)

dtc_freq (частота детекций)

clr_obs (число/уровень наблюдений в ясных условиях)

avg_temp (средняя температура сигнала)

ellip (параметр геометрии/неопределенности детекции)
